# Shared State UI

A Next.js prototype implementing an accessible "Shared State" AI interaction paradigm. The application uses Google Gemini (via Server Actions) to generate dynamic UI components based on user-supplied context and prompts.

## Project structure

- `shared-state-ui/` — Next.js application
- `docs/plans/` — implementation plan

## Quick start

```bash
cd shared-state-ui
npm install
npm run dev
```

Open http://localhost:3000 to use the app.

See `shared-state-ui/README.md` for full setup instructions.
